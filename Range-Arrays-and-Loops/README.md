# Instructions

Write a method that takes in a minimum and maximum integer, and return an array of all the integers in between in ascending order.

**Example**

```js
rangeGen(3, 7) => [4, 5, 6]
rangeGen(-4, 1) => [-3, -2, -1, 0]
```
