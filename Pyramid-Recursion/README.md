# Instructions

Using recursion, write a function that returns the total number of blocks in a pyramid with a given height. A pyramid looks like this:

```
row 1:     *
row 2:    ***
row 3:   *****
ROW 4:  *******
```

so the number of blocks for a row n equals `2 * n - 1`.

**Example**

```js
pyramid(3) => 9
```
